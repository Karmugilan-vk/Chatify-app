🚀 Production Deployed | 🌐 Live Application Available

## 📌 Overview
Chatify is a full-stack real-time chat application built using the MERN stack.  
It enables seamless communication between users with secure authentication, real-time messaging, and media sharing.

This application is designed with a focus on scalability, performance, and production-level architecture.

---

## ⚙️ Tech Stack

### 🖥️ Frontend
- React (Vite)
- CSS / Modern UI

### 🔧 Backend
- Node.js
- Express.js

### 🗄️ Database
- MongoDB Atlas

### ⚡ Real-Time Communication
- Socket.IO

---

## 🔧 Integrations & Services
- ☁️ Cloudinary – Image Upload & Storage  
- 📧 Resend – Automated Email Services  
- 🛡️ Arcjet – Security & Protection

## ✨ Key Features
- 🔐 Secure User Authentication (JWT with HTTP-only cookies)  
- 🔑 Password Hashing using bcrypt  
- 💬 Real-time Messaging using Socket.IO  
- 🟢 Online / Offline User Status  
- 🖼️ Image Sharing via Cloudinary  
- 📩 Automated Email Notifications  
- 🔄 RESTful API Architecture  
- 📱 Responsive & Clean UI

## 📦 Installation & Setup

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/your-username/chatify-app.git
cd chatify-app
```

### 2️⃣ Install Dependencies
```bash
cd backend
npm install

cd ../frontend
npm install
```

### 3️⃣ Setup Environment Variables
Create a `.env` file in backend:

```env
PORT=3000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
CLOUDINARY_API_KEY=your_key
RESEND_API_KEY=your_key
CLIENT_URL=http://localhost:5173
```

### 4️⃣ Run the Application
```bash
# backend
npm run dev

# frontend
npm run dev
```
